### Что НЕ реализовано:

1. **Публичный сайт casa.kz** (п. 5.15, 5.19) — нет публичных страниц: ипотека, новостройки, блог, о проекте, сервисы. Только CRM-кабинет за логином.

2. **Интеграция с NextBot** (п. 5.9) — нет webhook-эндпоинтов для приёма заявок из бота.

3. **Интеграция с банковскими API** (п. 5.22) — перспективное требование, архитектура готова.

### Что нужно проверить:

1. **Баг смены email** (п. 5.6) — API исправлен, нужно ручное тестирование.
2. **Парсинг CRM застройщиков** (п. 5.8) — есть импорт из Excel, нет автопарсинга по логину/паролю.
3. **Полное тестирование** (п. 5.13) — нужен полный прогон всех сценариев.
4. **Продажа доступов** (п. 5.23) — бэкенд готов, нет UI регистрации.
5. **Деплой на ps.kz** (п. 5.34) — конфиг готов, нужно проверить на сервере.
