# TZ_Casa_Pro — Техническое задание

## Структура папки

```
TZ_Casa_Pro/
├── ТЗ — Casa Pro (Дополнительное соглашение).md   ← ГЛАВНЫЙ ДОКУМЕНТ
├── README.md                                        ← Этот файл
│
├── 01_UI/                  ← UI-макеты от Заказчика (Canvas-описания + скриншоты)
│   ├── Dashboard Main Page.md + .png        — Главная страница
│   ├── Crm Deals Page.md + .png             — Сделки (CRM)
│   ├── Casa Pro Novostroyki V1 2.md + .png  — Новостройки
│   ├── Casa Pro Ipoteka V1 2.md + .png      — Ипотека
│   ├── Casa Pro Profile V1 2.md + .png      — Профиль брокера
│   ├── Копия Crm Lead Incoming Ui.md + .png — Форма нового контакта продавца
│   └── Casa Pro Object Card.md + .png       — Карточка объекта недвижимости
│
├── 02_Business/            ← Бизнес-документация от Заказчика
│   ├── CASA STRATEGY.md              — 12 стратегий продажи (S1–S12)
│   ├── Воронки "Продавец" "Объект".md — Две воронки CRM
│   ├── Данные продавца.md            — Структура данных продавца
│   ├── Объект недвижимости.md        — Параметры объекта
│   └── Услуга Продажа квартиры.md    — Описание услуги
│
└── 03_Current/             ← Текущая реализация (для сверки)
    └── BUSINESS_LOGIC.md   — Описание текущей бизнес-логики системы
```

## Как читать

1. Начните с главного документа: `ТЗ — Casa Pro (Дополнительное соглашение).md`
2. Каждый пункт ТЗ ссылается на конкретные файлы из папок 01_UI и 02_Business
3. Папка 03_Current содержит описание текущей реализации для сверки с требованиями
4. Все .png файлы — скриншоты UI-макетов, .md — текстовые описания к ним
